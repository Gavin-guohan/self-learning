# Java后端各模块Demo

Week 3-8 的后端技术栈练习代码。

## 目录结构

```
java_backend/
├── mybatis_demo/          # MyBatis / MyBatis-Plus 练习
├── spring_demo/           # Spring Framework 核心练习
├── springboot_demo/       # Spring Boot 实战练习
└── redis_demo/            # Redis + 中间件练习
```

## 技术栈
- Spring Boot 3.x
- MyBatis-Plus
- MySQL 8.x
- Redis
- Maven
