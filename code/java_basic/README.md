# Java基础练习代码

Week 1-2 的Java语言基础练习代码。

## 目录结构

```
java_basic/
├── basics/            # 数据类型、控制流、数组、方法
├── oop/               # 面向对象（类、继承、多态、接口）
├── collection/        # 集合框架练习
├── generics/          # 泛型
├── reflection/        # 反射
├── io/                # IO流、NIO
├── thread/            # 多线程与并发
└── jvm/               # JVM相关实验
```

## 运行方式

```bash
javac XxxDemo.java
java XxxDemo
```
